# Icons

This directory should contain the application icons.

## Required Icons

### macOS Menu Bar Icon (Tray Icon)
- `trayTemplate.png` - 16x16px or 32x32px PNG with transparency
- Use template naming convention for automatic dark/light mode support

### Application Icon
- `icon.png` - 512x512px PNG
- Used for DMG installer and application bundle

## Creating Icons

You can create icons using design tools like:
- Figma
- Sketch
- Adobe Illustrator
- or online tools like https://www.canva.com/

### Tips for Tray Icons
- Keep it simple and recognizable at small sizes
- Use template images (black with transparency) for best macOS integration
- File should be named with `Template` suffix for automatic theming
- Recommended size: 16x16px @1x, 32x32px @2x
